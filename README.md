## Setup
Make sure to install the dependencies:
```bash
npm install
```

## Development Server
Start the development server on `http://localhost:3000`
```bash
npm run dev
```

## Production
```bash
npm run build
```

[Nuxt Deploy docs](https://nuxt.com/docs/getting-started/deployment) for more information.
